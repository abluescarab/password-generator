Introduction

Thanks for using the Password Generator!

Features:

  • Choose what types of characters to use
  • Choose the length of your password
  • Disable character repeats
  • Save generated passwords to a text file
  • Mass generate passwords and save them to a text file

Generating a Password

To start using the Password Generator, just select your character sets and click Generate!

The statusbar at the bottom of the window will display important messages. Make sure to look at it after generating!

Options

There are plenty of options included in the Password Generator, all designed to make it as simple and full-featured as possible.

General Options

Do not use the same character twice

This will ensure that characters do not repeat (for example, if "A" is already in the password, do not use it again). If your password is long and not enough character sets are selected, you will receive an error message. Either select more characters or make your password shorter.

Length of password

This changes the length of the password. The maximum is 97 (one for each non-repeating character).

Characters

The checkboxes represent different character sets to use. If they are checked, they will be included in the password. You can choose uppercase and lowercase letters, numbers, and various special characters.

Save to Text File

This will allow you to save all the passwords you generate to a text file somewhere on your hard drive. Enabling this option will allow the use of mass generation.

Click the Browse... button to change the location that you save passwords to.

Mass Generation

This option will allow you to generate multiple passwords at once. It only becomes available if you enable saving to a text file.

Change the Generation amount to a number of your choosing. The maximum amount of passwords you can generate at once is 1000.
